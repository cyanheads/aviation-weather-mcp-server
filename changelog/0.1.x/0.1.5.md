# 0.1.5 — 2026-06-20

## Changed

- **`@cyanheads/mcp-ts-core` `^0.10.6` → `^0.10.9`** — picks up the `check-dependency-specifiers` devcheck step (hard-fails on floating `latest` / `*` / pre-release dist-tag specifiers in `package.json` and `bun.lock` workspaces), plugin-marketplace manifest validation in `lint:packaging`, fresh-scaffold devcheck robustness (changelog-sync and git-dependent checks self-skip outside a git repo), a `DuckdbProvider.describe()` binder fix on filtered table lookups, the new `ctx.content` collector for image/audio blocks, and the biome 2.5 ruleset migration.
- **`devcheck.config.json`** — added `packaging.pluginManifests: true`, enabling the upstream plugin-manifest checks against `.claude-plugin/plugin.json` and `.codex-plugin/`.
- **Vendored skills + scripts** — `skills/` and the `scripts/` devcheck toolchain resynced to the `@cyanheads/mcp-ts-core` `0.10.9` set; adds `scripts/check-dependency-specifiers.ts`.
