# 0.1.3 — 2026-06-06

## Fixed

- **`aviation_find_stations`** tool description and `station_ids` parameter description no longer claim IATA/FAA support — the AWC `stationinfo` endpoint returns HTTP 204 for non-ICAO IDs; `station_not_found` recovery hint updated to "use ICAO format (e.g., KSEA, not SEA)". ([#1](https://github.com/cyanheads/aviation-weather-mcp-server/issues/1))
- **`aviation_get_pireps`** `no_pireps_found` error now names the active altitude filter and reports how many PIREPs existed at other altitudes; recovery hint directs to `altitude_min_ft` / `altitude_max_ft`; falls back to the generic distance/hours hint when no altitude filter was active. ([#2](https://github.com/cyanheads/aviation-weather-mcp-server/issues/2))
- **`aviation_get_advisories`** `hazard` enum descriptions differentiated: `SURFACE WIND` = "sustained strong surface winds (typically >30 kt)"; `LLWS` = "low-level wind shear below 2,000 ft AGL" (previously both were described as "low-level wind shear"). ([#3](https://github.com/cyanheads/aviation-weather-mcp-server/issues/3))
